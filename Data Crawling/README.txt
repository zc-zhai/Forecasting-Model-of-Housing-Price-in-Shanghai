# lianjia.py 这是用来爬取链家网上海地区房源信息的python脚本

# 需要引用以下模块：
#               requests：是python实现的简单易用的HTTP库，主要用于请求目标网站
#               parsel：是Scrapy包中的解析模块，可以方便地在网页内容中选择到需要的部分。主要使用XPath来解析网页源码，获取信息
#               pandas：使用dataFrame数据结构进行数据的操作，写入
#               openpyxl：excel的支持库，不引入的话会报错...

# 每一条信息的包含以下内容，共28个特征：
#               title, 总价, 均价, 小区名, 区域1(所在区县), 区域2(区县下的城镇), 
				房屋户型, 所在楼层, 建筑面积, 户型结构, 套内面积, 建筑类型, 
				房屋朝向, 建筑结构, 装修情况,梯户比例, 配备电梯, 产权年限, 
				挂牌时间, 交易权属, 上次交易, 房屋用途, 房屋年限, 产权所属, 
				抵押信息, 房本备件, 特色标签, 卖点
# 共爬取到39415条数据，见house_info.xlsx
